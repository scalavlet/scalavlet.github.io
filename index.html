Scalavlet

Scalavlet is yet another tiny Sinatra like Scala web framework on top of Servlet specification. That provides a simple coding way for HTTP request routing without any XML configuration and complicated Scala implicit conversions. 


class SampleServlet extends Scalavlet {
  get("/ping") { request =>
    "pong"
  }
}

class Bootstrap extends ScalavletBootable {
  def onStart (context: Context): Unit = {
    context.mount(new SampleServlet, "/")
  }
}
